# Synthetic Tabular Data: генерация и оценка

Репозиторий для экспериментов по **синтетическим табличным данным**: генерация (CTGAN/TVAE и т.п.), тюнинг/дообучение генераторов и оценка качества синтетики через:
- **utility-метрики** (качество downstream-моделей: `acc`, `f1`, `roc_auc` и др.)
- **fidelity-метрики** (статистическая близость распределений)\

  
Реализован полный цикл обучения генеративных моделей, начиная с парсинга датасетов, заканчивая оценкой качества downstream-моделей при обучении на синтетических данных различных генераторов, а также реальных данных. Главной особенностью является доработка целевой функции генератора для повышения качества, состоящая в дополнительном слагаемом, оптимизирующем статистическую близость реальных данных и синтетики.
При этом рассмотрены различные downstream-задачи: классификация с балансом/дисбалансом классов, регрессионные задачи, наличие как категориальных, так и непрерывных признаков

## Быстрый старт

### 1) Окружение
Рекомендуется Python 3.10+.

```bash
python -m venv .venv
# Windows:
# .venv\Scripts\activate
# macOS/Linux:
# source .venv/bin/activate

pip install -r requirements.txt
```

### 2) Запуск
Откройте и выполните ноутбук:

- `notebooks/Синтетика_full.ipynb`

В самом ноутбуке маркдаунами отмечены основные этапы пайплайна. Артефакты будут появляться по мере выполнения кода ячеек

Ключевые артефакты сохранены в следующих местах:
- таблицы метрик → `results/tables/`
- графики → `results/plots/`
