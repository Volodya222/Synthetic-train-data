# Synthetic Tabular Data: генерация и оценка

Репозиторий для экспериментов по **синтетическим табличным данным**: генерация (CTGAN/TVAE/SDV и т.п.), тюнинг/дообучение генераторов и оценка качества синтетики через:
- **utility-метрики** (качество downstream-моделей: `acc`, `f1`, `roc_auc` и др.)
- **fidelity-метрики** (статистическая близость распределений; при необходимости добавляются отдельными таблицами/графиками)

## Структура проекта

```
.
├── notebooks/
│   └── Синтетика_full.ipynb          # основной ноутбук: пайплайн генерации/оценки
├── results/
│   ├── tables/                       # таблицы метрик/сводки
│   │   └── utility_metrics_example.csv
│   └── plots/                        # сохранённые графики (папка под артефакты)
├── data/                             # данные (лучше не коммитить большие файлы)
├── src/                              # опционально: вынести функции/утилиты из ноутбука
├── requirements.txt
└── README.md
```

## Быстрый старт

### 1) Окружение
Рекомендуется Python 3.10+.

```bash
python -m venv .venv
# Windows:
# .venv\Scripts\activate
# macOS/Linux:
# source .venv/bin/activate

pip install -r requirements.txt
```

### 2) Запуск
Откройте и выполните ноутбук:

- `notebooks/Синтетика_full.ipynb`

В процессе выполнения сохраняйте ключевые артефакты:
- таблицы метрик → `results/tables/`
- графики → `results/plots/`

## Как добавлять новые результаты

1. Сохраните таблицу (`.csv`/`.xlsx`) в `results/tables/`
2. Сохраните графики (`.png`/`.pdf`) в `results/plots/`
3. При необходимости добавьте в README краткое описание: что за эксперимент и какой вывод.
